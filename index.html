AI Vision Analysis

Proprietary multimodal prompts for Google Gemini optimized to recognize Ladder, FBD, and functional diagrams with high precision.

IEC 61131-3 SCL

Generates industrial standard Structured Text (SCL). Includes full block headers, variable declarations, and logic body.

Tag Synchronization

Generates PLC Tag Excel tables (`.xlsx`) and keeps them 100% synced with the code to avoid compilation errors.

How to use locally

01

Clone the repository and install dependencies using pip install -r requirements.txt.

02

Create a .env file with your GEMINI_API_KEY or enter it directly in the UI.

03

Run start.bat to launch the server and go to localhost:5000.
